# Motion Map

## Purpose
This file maps motion intensity and ownership by page family and section type.

## Motion intensity scale
- Level 0: static / utility-first
- Level 1: subtle reveal / hover
- Level 2: coordinated stagger / section entrance
- Level 3: immersive identity storytelling

## Page motion policy

### Home
Allowed intensity:
- Level 2 to Level 3
Typical uses:
- intro reveal
- section pacing
- image/text sequencing
- premium transitions
Avoid:
- long blocking intros before work becomes visible

### Work listing
Allowed intensity:
- Level 1 to Level 2
Typical uses:
- card reveal
- hover motion
- filter/sort transition if present
Avoid:
- heavy pinning
- effects that slow scanning

### Project detail
Allowed intensity:
- Level 1 to Level 2
Typical uses:
- hero reveal
- image sequencing
- section transitions
Avoid:
- excessive decorative motion that distracts from the case study

### About
Allowed intensity:
- Level 1 to Level 2
Typical uses:
- portrait reveal
- staggered copy entrance
- subtle scrolling rhythm
Avoid:
- overcomplicating personal storytelling with spectacle

### Footer / contact
Allowed intensity:
- Level 0 to Level 1
Typical uses:
- subtle link hover
- light reveal
Avoid:
- anything that competes with the CTA

## Reduced motion policy
- content must remain readable without animation
- identity should still come through with reduced motion enabled
- no essential navigation should depend on timing or transitions
